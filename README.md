# niyanta

A self-orchestrated, multi-tenant ingestion platform built on a durable distributed activity engine. Customers register connector connections and ingestion policies; Niyanta plans, executes, observes, adapts, and recovers ingestion runs across dense shared worker pools while preserving tenant isolation.
